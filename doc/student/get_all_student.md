# Consulta de todos os clientes

Retorna todos os clientes matriculados na academia.

- **PATH** <br />
  /student

- **Method** <br />
  `GET`

- **Success Response**

  - **Code**: 200 <br />
    **Description**:
    ```json
    [
      {
        "id": 1,
        "name": "xxxx",
        "email": "xxxx@xxxx.xxx",
        "avatar": null
      }
    ]
    ```
